# Publish items demo

## Requirements

To run the demo, you need the ff:

* Access to a Shopify Plus store
* Shopify app with access token and write_publication permissions. You _might_ need to reach out to Shopify Support to enable this.

## Running

``` shell
$ yarn
$ yarn start
```
